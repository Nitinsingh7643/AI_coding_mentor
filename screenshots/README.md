# Screenshots Guide

This directory contains screenshots of the AI Coding Mentor application.

## How to Add Screenshots

1. **Start the application**:
   ```bash
   npm start
   ```

2. **Navigate to each page** and take screenshots:

### Required Screenshots:

#### 1. **home-page.png**
- URL: `http://localhost:3000`
- What to capture: Full landing page including:
  - Hero section with "Supercharge Your Coding Skills with AI"
  - Statistics (10,000+ Developers, 25+ Languages, 98% Satisfaction)
  - Features grid
  - Testimonials section
  - Pricing cards

#### 2. **main-interface.png**
- URL: `http://localhost:3000/app`
- What to capture: Main application interface showing:
  - Code editor with sample code
  - Mode selector (Improve/Explain/Comment/Debug)
  - "Analyze Code" button
  - Empty results section

#### 3. **code-analysis-result.png**
- URL: `http://localhost:3000/app`
- What to capture: After analyzing code:
  - Original code in editor
  - AI analysis panel with suggestions
  - Improved code display
  - Complexity metrics (Time & Space)
  - Diff viewer showing changes

#### 4. **dashboard.png**
- URL: `http://localhost:3000/dashboard`
- What to capture: Dashboard page showing:
  - Analytics and statistics
  - Recent code analyses
  - Usage metrics

#### 5. **features-section.png**
- URL: `http://localhost:3000` (scroll to features)
- What to capture: Close-up of the features grid showing:
  - Code Refactoring
  - Code Explanation
  - Smart Comments
  - Voice Coding
  - Debugging Assistant
  - Learning Paths

#### 6. **pricing-section.png**
- URL: `http://localhost:3000` (scroll to pricing)
- What to capture: Pricing cards showing:
  - Starter plan ($9/month)
  - Developer plan ($29/month) - highlighted
  - Team plan ($99/month)

## Screenshot Specifications

- **Format**: PNG (preferred) or JPG
- **Resolution**: At least 1920x1080 for desktop views
- **Mobile**: Add mobile screenshots (375x667) with `-mobile` suffix
- **Naming**: Use lowercase with hyphens (e.g., `home-page.png`)
